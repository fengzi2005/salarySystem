## 项目启动与基础搭建

### 需求理解

- 向AI提交《工资管理系统需求文档（按功能模块划分）.docx》
- AI提出确认问题：
  - Spring Boot版本选择（降级到3.x稳定版）
  - 基本工资≠岗位工资（二者独立存储、分开核算）
  - 登录方案（JWT+Redis+拦截器）
  - UI风格（蓝白配色、左侧导航）

### 技术选型决策

- Spring Boot 3.4.1（从4.1.0 pre-release降级）
- MyBatis Plus 3.5.5（从3.5.9修正，因3.5.9不存在）
- Element Plus 2.9.1 + ECharts 6
- MySQL密码修正：原文档写123456，实际为123

### 数据库设计

- 设计15张表的完整schema（01-schema.sql）
- 编写8个存储过程 + 2个触发器 + 2个视图 + 1个定时事件
- 准备26名员工、8个部门、10个岗位、5个职称的初始数据

### 后端搭建

- 56个Java文件
- 分层架构：common → config → entity → mapper → service → controller → interceptor
- JWT工具类 + UserHolder(ThreadLocal) + 登录/令牌拦截器
- 修复MyBatis Plus编译错误：
  - StatementType导入修正
  - DepartmentServiceImpl list(null)歧义修正
  - SeniorityPayRuleServiceImpl LambdaUpdateWrapper修正

### 前端搭建

- Vue 3 + Element Plus + Vite配置
- 13个功能页面骨架
- Axios封装（Token注入 + 401拦截重定向）
- 路由守卫 + Pinia状态管理
- Vite代理配置解决CORS

## 联调测试与收尾

### 测试发现的问题

1. **工号重复**：后端增加LambdaQueryWrapper查重
2. **考勤重复**：前端增加checkExisting自动转编辑模式
3. **工资项启停无效**：存储过程未读取salary_item_config → 增加12个is_active判断
4. **已确认被覆盖**：存储过程增加CONFIRMED跳过逻辑
5. **孤儿数据**：删除员工时关联数据残留 → 级联删除
6. **Redis不可用**：角色信息丢失 → JWT拦截器先解析Token再查Redis
7. **首页报表异常**：Promise.all一个失败全部reject → 个人薪资改try-catch单独处理
8. **邮箱校验过严**：fosu.edu.cn不通过 → 域名部分允许点号
9. **排序分页冲突**：服务端分页后再排序导致跨页数据遗漏 → 全量加载+客户端排序

### 定时任务测试

- MySQL Event状态验证（ENABLED）
- event_scheduler确认ON
- 手动调用存储过程模拟自动计算 → 27人全部成功

## 关键技术决策记录

| 决策     | 选项A       | 选项B        | 最终选择       | 原因                         |
| ------ | --------- | ---------- | ---------- | -------------------------- |
| 密码存储   | BCrypt加密  | 明文         | 明文(演示)     | 演示环境简化                     |
| 工资计算   | Java逐条计算  | 存储过程       | 存储过程       | 减少网络往返、原子性                 |
| 分页方式   | 服务端分页     | 客户端分页      | 客户端分页      | 需全局排序后分页                   |
| 权限粒度   | 仅角色       | 角色+岗位+部门   | 三者结合       | 需求精确到岗位级别                  |
| 工号校验   | 仅前端       | 前端+后端      | 前后端双重      | 保证数据唯一性                    |
| 字段命名   | camelCase | snake_case | snake_case | 后端JdbcTemplate返回snake_case |
| E-R图工具 | Mermaid   | Graphviz   | Mermaid    | 无需安装、浏览器渲染                 |
